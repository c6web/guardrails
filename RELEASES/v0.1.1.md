# Release v0.1.1

**Date:** 2026-07-08

**C6 Guardrails — First Release.**

This is the initial public release of C6 Guardrails, an AI Firewall Gateway that proxies and protects AI application traffic against the OWASP LLM Top 2025 threat landscape.

## What's Included

- **LLM Proxy Gateway** — high-performance Rust-based proxy supporting OpenAI, Anthropic, Gemini, Ollama, and OpenRouter providers
- **Threat Detection** — keyword/regex pre-filtering + LLM-powered classification against OWASP LLM Top 2025 threats
- **Management Console** — React-based dashboard for managing policies, detectors, ACLs, apps, API keys, and audit logs
- **Policy Engine** — configurable block/redact/flag enforcement with fail-open and fail-closed modes
- **Semantic Cache** — embedding-based response caching to reduce latency and cost
- **Content Quality** — configurable content quality evaluation
- **Tool Guardrails** — governance rules for tool/function call usage
- **Rate Limiting & Quotas** — per-app and per-provider usage controls
- **Audit Logging** — comprehensive request, admin, and user activity auditing
- **Multi-language SDKs** — Node.js and Python SDKs for easy integration
- **Docker-based Deployment** — single-node demo or multi-service production setups
